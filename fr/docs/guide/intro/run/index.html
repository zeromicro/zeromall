服务启动顺序:​

服务启动顺序:

  1. 中间件服务: 配置中心, 服务注册中心, 服务发现中心, 服务网关, 服务监控中心
  2. 基础设施服务: 短信服务, 邮件服务, 推送服务
  3. 核心服务: 帐号系统, 登录系统, 权限系统

服务定级标准:

  1. 服务的重要性: 基础中间件先启动, 短信服务, 邮件服务, 推送服务
  2. 核心服务: 帐号系统, 登录系统, 权限系统
