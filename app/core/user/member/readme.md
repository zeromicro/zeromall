# user/member:

- 账号系统, 支持各种第三方登录方式

## 功能:

- 用户基本的 CRUD.
- 用户设置更改:
    - 密码修改
    - 手机号修改
    - 邮箱修改
    - 用户基础信息修改
    - 其他设置
