ZeroMall 使用指南

ZeroMall: B2B2C 电商平台

  • 基于: Go + go-zero + gRPC 微服务开发.
  • 集成中间件: mysql, redis, kafka, consul 等.
  • API: http rest/grpc/websocket.

eShop

  • eShop:小型电商官网.
  • 集成授权码/license 售卖服务.
  • 集成微信/支付宝/数字货币等支付工具.
